Определение. Пусть R − векторное пространство над R с базисом
1 = (1 0)
i = (0 1)
Зададим умножение в R2 таблицей умножения в этом базисе:
* 1 i
1 1 i
i i -1

Теорема. Указанная алгебра является полем. Это поле называется полем комплексных чисел. При этом умножение задается формулой:
(x + iy)(x' + iy') = (xx' − yy')+(xy' + x'y)i
(x, y ∈ R)
C = (R, +, *) - поле

Доказательство. Знаем, что C − алгебра над R. Надо проверить коммутативность умножения, ассоциативность умножения, существование единицы, существование обратного для ненулевого элемента
1) Коммутативность следует из равенства 1 · i = i · 1 и теоремы об алгебрах
2) Ассоциативность проверим для элементов базиса 1 и i
Случай 1) (ab)c = a(bc), один из элементов − это 1
Тогда (1 · b)c = 1 · (bc) ⇔ bc = bc
(a · 1)c = a(1 · c) ⇔ ac = ac
(ab) · 1 = a(b · 1) ⇔ ab = ab
Случай 2. a = b = c = i
(i · i) · i = i · (i · i) ⇔ −i = −i
По теореме об алгебрах ассоциативность умножения верна для всех элементов
алгебры.
3) Существование единицы. Очевидно, что элемент 1 = (1 0)
4) Существование обратного. Пусть α + βi != 0, α, β ∈ R ⇔ (a, b) != (0, 0) ⇔ α^2 + β^2 > 0.
Элемент α − βi называют сопряженным α + βi
Имеем (α + βi)(α − βi) = α^2 − (βi)^2 = α^2 + β^2 > 0.

Разделим это неравенство на α^2 +β^2 . Тогда
(α + βi)(α − βi) / (α^2 + β^2) = 1
(α + βi)^-1 = (α − βi)/(α^2 + β^2)
