Теорема о кратных корнях для мн-на и производной:
- Пусть K - поле, c ∈ K, f(x) ∈ K[x]
- k = 1+…1[k раз] ≠ 0 в поле K
- Если c - k-кратный корень f(x), то c - (k-1) кратный корень f'(x)
--ДОК-ВО--
- Пусть f(x) = (x-c)^k * g(x); g(c) ≠ 0; g(x) ∈ K[x]
- Тогда:
- f'(x) = k(x-c)^[k-1] * g(x) + (x-c)^k * g'(x) = 
- = (x - c)^[k-1] * (kg(x) + (x-c)g'(x))
- 2я скобка ≠ 0

--END--
