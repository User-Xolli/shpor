Алг сис A = 〈A; ∩,∪, a, 0, 1〉 наз-ся булевоевой алгеброй, если для люб a, b, c из A выполняются следующие условия:
1) a ∪ b = b ∪ a
2) a ∩ b = b ∩ a
3) a ∪ (b ∪ c) = (a ∪ b) ∪ c
4) a ∩ (b ∩ c) = (a ∩ b) ∩ c
5) a ∪ (b ∩ c) = (a ∪ b) ∩ (a ∪ c)
6) a ∩ (b ∪ c) = (a ∪ b) ∩ (a ∪ c)
7) ¬(a ∪ b) = ¬a ∩ ¬b
8) ¬(a ∩ b) = ¬a ∪ ¬b
9) a ∪ a = a
10) a ∩ a = a
11) ¬¬a = a;
12) a ∪ ¬a = 1;
13) a ∩¬a = 0;
14) a ∪ 0 = a;
15) a ∩ 0 = 0;
16) a ∪ 1 = 1;
17) a ∩ 1 = a.
