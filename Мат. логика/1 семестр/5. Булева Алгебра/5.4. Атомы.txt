Атомом называется мин среди ненулевых элементов булевой алгебры. Т. е. a - атом, если a != 0 и для люб b ∈ A из 0 <= b <= a следует (b = 0 либо b = a)
----
Элемент b булевой алгебры называется безатомным, если под ним нет атомов, т. е. если c <= b, то c – не атом.
----
Элемент c булевой алгебры называется атомным, если под ним нет ненулевых безатомных элементов, т. е. если b ≤ c и b – безатомный, то b = 0
----
Булева алгебра называется атомной, если в ней нет безатомных элементов, кроме нуля. Булева алгебра называется безатомной, если в ней нет атомов
----
Булева алгебра A называется вырожденной, если 0 = 1
----
Для любого множества алгебра всех его подмножеств является атомной. Если X – множество и B = 〈p(X),∪,∩, −, ∅, X〉 явл булевой алгеброй всех его подмножеств, то
1) At(B) = {{a}| a ∈ X}
2) B – атомная булева алгебра
--ДОКВО---
1) (<=) Пусть A ∈ At(B). Тогда A != Ø. Следовательно, существует элемент a ∈ A. Тогда {a} != Ø и {a} <= A. Значит, {a} = A
(>=) Рассмотрим a ∈ A. Очевидно, что {a} != Ø. Пусть B <= {a} и
B != Ø. Следовательно, найдётся такой элемент b ∈ B, что b ∈ {a}.
А значит, b = a. Следовательно, {a} <= B, и тогда B = {a}. Таким
образом, получили, что {a} – атом.
-
2) Рассмотрим множество A <= X, A != Ø. Тогда сущ эл-т a ∈ A. Следовательно, {a} <= A и {a} − атом. Значит, B = 〈ρ(X),∪,∩, −, ∅, X〉 – атомная булевая алгебра
