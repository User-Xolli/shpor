--Предлож
Пусть B ∈ K(σ), X ⊆ |B|, X ≠ ∅
Тогда
∃ подмодель C ≤ B, кот-я явл. наименьшей по вкл. среди таких подмоделей A ≤ B: X ⊆ |A| (среди всех подмоделей, содерж. X, ∃ наименьшая подмодель).
Эта подмодель наз-ся подмоделью, порождаемой мн-ом X

C = sub_B(X)
--ДОК-ВО
Рассм. H = { A≤B | X ⊆ |A|} ≠ ∅, т.к. B ∈ H ⇒ H ≠ ∅ .

Тогда C = [A∈H]_∩|A|,
X ⊆ C, X ≠ ∅ ⇒ C ≠ ∅ ⇒
∃ ζ ≠ B: |ζ| = C
⇒X ⊆ |ζ|
Кроме того, если A≤ζ,
X ⊆ |A|, то A ∈ H ⇒ C ⊆ |A| ⇒ |C| ⊆ |A|
∎
--END
