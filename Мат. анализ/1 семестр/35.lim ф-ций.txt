- Опр. пред. ф-ции по Коши и по Гейне.
- Опред. односторон. пред.
- Т. о связи односторон. пред. с двусторон(б/д)
----
- Пусть задано мн-во X.
- Под окрестностью точки a будем понимать люб. интервал вида (a−δ, a+δ)
- Точка a наз-ся точкой сгущ. мн-ва X, если в любой ее окрестности содерж. по кр. мере одна точка из X, отл. от a.
- a может как принадлежать X, так и не принадлежать
---
---Опр. предела по Коши---
- Пусть a — точка сгущ. мн-ва X
- Пусть f(x) задана на X
- Число A наз-ся пределом ф-ции f(x) при стремлении x к a,
- если для люб ε > 0 сущ δ(ε,a)
- т.ч. из 0 < |x − a| < δ =>
- |f(x) − A| < ε
- lim_[x→a](f(x)) = A
---
--Опр. предела по Гейне--
- Если a — точка сгущ.,
- то в люб. окрестности точки a содерж. беск число точек мн-ва X =>
- мы можем бесконеч. числом способов обр. из точек X послед.
- {xn}, кот-ая сход. к a ({xn} → a)
- {xn} соотв. послед. f (x1)…f(xn)…
- Если люб {xn} → a
- lim_[n→∞](f(xn)) = A,
- то A наз-ся пределом f(x) при x→a
- lim_[x→a]f(x) = A
---
Число A наз-ся пределом f(x) при стремлении x→a справа (слева), если люб ε > 0 сущ δ(ε,a) т.ч. из a < x < a+δ (a−δ < x < a) вытек. |f(x) − A| < ε
---
Для того, чтобы A явл. пред. f(x) при x→a необход. и достаточн.,
- чтоб левый и правый пределы совп.
--END--
