сущ:
Любая сист вложенных отрезков содержит хотя бы 1-но число. Это непрерывность (плотность) вещественных чисел по Кантору.
---
ед:
Будем говорить, что у сист влож отрезков длина стремится к нулю при возрастании параметра n, если люб ε > 0 сущ n(ε) т.ч. (b_n − a_n) < ε люб n > n(ε)
-
Пусть сист {[a_n, b_n]} удовл св-ву из пред опред. Тогда сущ ед. число, принадлежащее всем числовым промежуткам
---
ДОК-ВО:
- Пусть сущ 2 таких числа x и y, где для определенности x < y
- люб n a_n <= x < y <= b_n -> y−x < b_n−a_n
- Известно, что люб ε > 0 сущ n(ε) т.ч. b_n−a_n < ε при n > n(ε) -> y − x < ε люб ε > 0
- Т.к. x < y -> сущ ε(0) т.ч. y−x = ε(0). Но y − x < ε люб ε > 0 -> и для ε = ε(0) -> ε(0) = y − x < ε(0). Противоречие
---END---
