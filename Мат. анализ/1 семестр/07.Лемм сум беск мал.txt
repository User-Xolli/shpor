Лемма о сум беск мал
--
Сумма любого конечного числа беск. малых беск. мала
---
ДОК-ВО:
- Пусть {an} и {bn} — беск. малые послед
- (limn→∞(an) = 0, lim_n→∞(bn) = 0)
- По люб заданному ε сущ номер N_1(ε) т.ч. |an| < ε/2, n > N_1(ε).
- сущ N_2(ε): |bn| < ε/2, n > N_2(ε) =>
- |an + bn| <= |an| + |bn| < ε/2+ε/2 = ε
- n > max{N1, N2}
--END--
